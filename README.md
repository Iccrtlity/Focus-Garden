# 🌿 Focus Garden — Browser Extension

Focus Garden is a minimalist productivity extension for Chrome that combines the Pomodoro technique with a virtual garden.

## ✨ Features
- **Precise Focus Timer:** A Pomodoro countdown.
- **Persistent Sessions:** Timer stays accurate even if closed.
- **Virtual Garden:** Progress visualization (🏜️ -> 🌱 -> 🌿 -> 🌳).
- **Customization:** Set your duration via settings.

## 🛠 Tech Stack
- Framework: React + TypeScript
- Build Tool: Vite
- Styling: Tailwind CSS
- Icons: Lucide React

## 🚀 Installation
1. Install dependencies: `npm install`
2. Build: `npm run build`
3. Load the `dist` folder in Chrome Extensions.
